# React CSV display in data table

This is a utility to put local csv data into react data table 

The [data](public/data/data.csv)  is displayed in a react carbon data table.

## How to run?

* ` Clone this repo `
* ` npm install `
* ` npm start `
* ` Go to localhost:3000 `
